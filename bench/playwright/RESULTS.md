## Caveats

- Playwright cold-starts Chromium per invocation of the sidecar (one process for all targets).
  heso is a one-shot binary; the suite reuses a single `reqwest` client across targets. Apples-to-apples for "extract title from N URLs in one run."
- httpbin.org is occasionally slower on heso because Playwright sometimes hits a warmer keep-alive.
  Run the suite a few times if you want a tighter spread.
- These numbers are total wall-clock including the network. Pure engine cost (what scales when you swap in cassettes) is the `ms_eval` column in each JSON.
